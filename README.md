# **Task Manager CLI**

A simple **command-line task manager** built with Python and SQLite.

---

## 🔹 **Features**
- ✅ Add new tasks
- 📜 View all tasks
- ✅ Mark tasks as **completed**
- ❌ Delete tasks
- 🛠 Uses **SQLite** for storing tasks
